INSTRUCCIONES PARA CREAR LA COLECCIÓN USERS EN FIREBASE

1. Ve a: https://console.firebase.google.com/u/0/project/micerebroapp/firestore/databases/(default)/data

2. Haz clic en "Crear colección"

3. Nombre de colección: users

4. Crea el primer documento con ID: 0HNCbSTfdlQN2q7vI9Fbq1flgnm1

5. Agrega estos campos al documento:
   - createdAt: timestamp = 9 de diciembre de 2025 a las 2:45:46 p.m. UTC-3
   - displayName: string = "Invitado"
   - email: null
   - emailVerified: boolean = false
   - photoURL: null
   - providerId: string = "anonymous"
   - uid: string = "0HNCbSTfdlQN2q7vI9Fbq1flgnm1"
   - lastLoginAt: timestamp = 9 de diciembre de 2025 a las 2:45:46 p.m. UTC-3"

6. Dentro del documento users/0HNCbSTfdlQN2q7vI9Fbq1flgnm1, crea una subcolección llamada: canvasBoards

7. En canvasBoards, crea un documento con ID: 1MjfoyyRNobXbFvKtVXP

8. Agrega estos campos al tablero:
   - id: string = "1MjfoyyRNobXbFvKtVXP"
   - name: string = "Mi Primer Tablero"
   - description: string = "Tablero de ejemplo para usuario invitado"
   - ownerId: string = "0HNCbSTfdlQN2q7vI9Fbq1flgnm1"
   - createdAt: timestamp = 9 de diciembre de 2025 a las 2:45:46 p.m. UTC-3
   - updatedAt: timestamp = 9 de diciembre de 2025 a las 2:45:46 p.m. UTC-3
   - isPublic: boolean = false
   - password: null

9. Dentro del tablero, crea una subcolección llamada: canvasElements

10. En canvasElements, crea un documento con ID: welcome-element

11. Agrega estos campos al elemento:
    - id: string = "welcome-element"
    - type: string = "text"
    - content: string = "¡Bienvenido a Mi Cerebro!"
    - position: map = {x: 100, y: 100}
    - size: map = {width: 300, height: 100}
    - style: map = {
        fontSize: "24px",
        color: "#000000",
        backgroundColor: "#ffffff"
      }
    - createdAt: timestamp = 9 de diciembre de 2025 a las 2:45:46 p.m. UTC-3
    - updatedAt: timestamp = 9 de diciembre de 2025 a las 2:45:46 p.m. UTC-3
